# Website Assets Archive
[standing-man (1).jpg]
This repository contains a collection of PNG images and GIF animations for websites. All files are ready to be copied and used in any web project.

## What's Inside

### PNG Images

PNG image files. These are static images that do not move or animate. Each PNG file is a single frame. The files come in various sizes and color styles including light mode versions and dark mode versions. Some PNG files have solid color backgrounds. Other PNG files have white backgrounds. Some PNG files have black backgrounds. Many PNG files have colored backgrounds. No PNG files in this archive have transparent backgrounds. Every PNG background is filled with a solid color.

### GIF Animations

GIF animation files. These are moving images that loop continuously. Each GIF file contains multiple frames that play in sequence to create motion. The animations run at various speeds. Some GIFs animate quickly. Other GIFs animate slowly. Some GIFs have smooth motion. Other GIFs have choppy or pixelated motion which is normal for the GIF format. Each GIF file loops endlessly when opened in a web browser.

## How to Use These Files

Using a PNG file requires copying the file into your website folder. After copying the file you can display it on your webpage using an img tag. You can also use a PNG file as a background image using CSS.

Using a GIF file works the same way. Copy the GIF file into your website folder. Then reference it with an img tag or as a CSS background image. The animation will start automatically when the page loads. The animation will loop forever without stopping.

## File Properties

PNG files in this archive preserve full image quality. They do not lose detail when saved. They support millions of colors. They are larger in file size compared to JPEG but smaller than BMP.

GIF files in this archive are limited to 256 colors per frame. This gives them a retro or pixelated look. They have smaller file sizes than PNG for simple animations. They do not support as many colors as PNG. The dithering effect is normal for this format.

## Where to Place Files

Put all PNG and GIF files in an assets folder inside your website directory. Or put them in an images folder. Or put them in a media folder. Or put them directly in the root folder. The location does not matter as long as your HTML code points to the correct path.

## Compatibility

PNG files work in every web browser ever created. GIF files also work in every web browser ever created. No special plugins are needed. No JavaScript is required. No CSS is needed for the files to display or animate. The GIF animation plays automatically without any code other than an img tag.

## File Naming

Each file has a unique name that describes what the image or animation looks like. The names use lowercase letters and numbers. Spaces are replaced with underscores. The file extension at the end indicates whether it is PNG or GIF.
